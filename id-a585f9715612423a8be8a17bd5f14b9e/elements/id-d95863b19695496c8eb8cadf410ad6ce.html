Gedragselementen uit de kern ()
EN: Behaviour elements represent the dynamic aspects of the enterprise. Similar to active structure elements, behaviour elements can be subdivided into internal behaviour elements and external behaviour elements; i.e., the services that are exposed to the environment. An internal behaviour element represents a unit of activity that can be performed by one or more active structure elements. An external behaviour element, called a service, represents an explicitly defined exposed behavior. NL: Gedragselementen vertegenwoordigen de dynamische aspecten van de organisatie. Net als bij actieve structuurelementen kunnen gedragselementen worden onderverdeeld in interne gedragselementen en externe gedragselementen; dat wil zeggen, de diensten die worden blootgesteld aan de omgeving. Een intern gedragselement vertegenwoordigt een activiteitseenheid die kan worden uitgevoerd door een of meer actieve structuurelementen. Een extern gedragselement, een service genaamd, vertegenwoordigt een expliciet gedefinieerd zichtbaar gedrag.
