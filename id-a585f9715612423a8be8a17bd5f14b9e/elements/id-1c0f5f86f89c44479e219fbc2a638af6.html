Informatie- en applicatielaag
EN: The information and application layer explains how the information system can be logically organised. Application services support work processes and are accessible via application interfaces. The application interfaces serve a business role so that the functionality from the application service is made available to the business role via the application interface. Realise application functions or operate application services. Application functions are assigned to application components. Application functions read and/or write data objects that are the information system instance of a business object and, in that capacity, realise business objects. NL: De informatie- en applicatielaag licht toe hoe het informatiesysteem logische ingericht kan worden. Applicatieservices ondersteunen de werkprocessen en zijn toegankelijk via applicatie interfaces. De applicatie interfaces bedienen een bedrijfsrol, zodat de functionaliteit uit de applicatieservice via de applicatie interface aan de bedrijfsrol beschikbaar wordt gesteld. Applicatie functies realiseren of bedienen applicatieservices. En applicatie functies zijn toegewezen aan applicatiecomponenten. Applicatie functies lezen en/of schrijven data objecten die de informatiesysteem instantie zijn van een bedrijfsobject en in die hoedanigheid bedrijfsobjecten realiseren.
